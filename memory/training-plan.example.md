## Periodization overview
_Use base → build → peak → taper for a dated event goal. For an ongoing/non-event goal,
replace this with a rolling focus-block structure instead (e.g. alternating volume and
strength/durability blocks) — periodization toward a fixed date doesn't apply._
- **Base** — aerobic volume, climbing durability, consistency.
- **Build** — add threshold/VO2 and long climbing-specific rides.
- **Peak** — race-specific intensity, sharpen.
- **Taper** — reduce volume, keep intensity, arrive fresh.
