# Bike Maintenance

_Service history and due/overdue tracking per bike. "Since last service" is computed by
comparing the bike's current total odometer (from the Strava MCP gear data) against the
odometer recorded at each item's last service — don't track a separate running total by
hand. Copy this file to `bike-maintenance.md` and fill it in._
